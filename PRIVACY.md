# Privacy Policy

Last updated: May 17, 2026

Invoice Helper for Bokio is a browser extension that runs on Bokio invoice editing pages to help select the intended invoice delivery option.

## Data Collection

Invoice Helper for Bokio does not collect, sell, transfer, or share personal user data.

The extension does not send data to any server, analytics service, advertising network, or third party.

## Local Settings

The extension uses `chrome.storage.local` to save one local preference: whether the user wants the extension to automatically select email delivery.

This setting stays on the user's device. It is not transmitted outside the browser.

## Website Access

The extension requests access to `https://app.bokio.se/*` so it can run on Bokio invoice pages and select invoice delivery options in the page interface.

The content script is limited to Bokio invoice editing pages:

```text
https://app.bokio.se/*/invoicing/invoices/edit/*
```

The extension reads and interacts with the page only as needed to provide its invoice delivery helper feature. It does not collect or transmit website content.

## Remote Code

Invoice Helper for Bokio does not load or execute remote JavaScript or WebAssembly. All executable extension code is included in the extension package.

## Limited Use

Any data handled by the extension is used only to provide or improve the extension's single purpose: helping users choose the intended Bokio invoice delivery option.

The extension does not use or transfer user data for unrelated purposes, advertising, creditworthiness checks, or lending decisions.
